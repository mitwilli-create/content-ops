# YouTube Playbook

length_window: null

**Role in the system:** Depth video + search-durable library. The only platform where content appreciates for years (search + suggested). Audiences 2-4. His on-camera + edit skills remove the usual creator bottleneck.

## Format split
- **Longform (8-15 min):** how-to and explainer library. Search-first titles ("How to build an AI memory file (so you stop repeating yourself)"). These earn for years.
- **Shorts (< 60s):** discovery feeder, repurposed from TikTok masters (export without watermark).
- **Live/streamed (later):** #18 family-consult format could be a live series once subscriber base exists.

## What works
- Chaptered, tightly-edited screen+face how-tos; retention graph is the whole game — cut every second that doesn't earn its place (he knows this from broadcast).
- Thumbnail + title as a unit: one concrete promise, readable at 120px. Test 2-3 thumbnail concepts per video.
- First 30 seconds: state the payoff and SHOW the end result, then rewind to steps.
- Description: timestamp chapters, links to Substack + repo; pinned comment carries the CTA.

## Timing & cadence (BASELINE Jan 2026 — /timing-check)
- Weekly upload day consistency > specific hour; suggested-feed dominates over subscription-feed anyway.
- Publish 2-4h before target audience's evening window; weekends fine for how-to content.

## Ledger fits
- Longform: #10 memory files, #15 skills/MCPs/tools, #16 memory deep-dive, #13 field-shoot multimodal (his flagship credential piece), #5 data centers (documentary-style, journalist register)
- Shorts: atomized from all of the above + TikTok masters
- #18 live consults: the franchise format. Start recorded + edited, go live later.

## Cross-platform sequencing
Substack essay (week 1) → YT longform of same idea (week 2) → 3-5 Shorts/TikToks atomized (week 2-3) → X thread with YT link once video has 48h of native traction.
