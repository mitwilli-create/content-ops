# Freshness discipline

All timing, algorithm, and platform-mechanics claims in knowledge/platforms/*.md are BASELINES dated 2026-01 (model knowledge cutoff). Platform metas shift monthly.

**Rule:** never schedule-recommend or assert a platform mechanic from the playbook alone. Run /timing-check (live WebSearch + Grok pulse) first; update the playbook file with the finding + date so baselines self-heal over time.

**Why:** stale timing advice presented confidently is worse than none; it also violates the freshness-first protocol Mitchell runs everywhere else.
