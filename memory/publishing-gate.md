# Publishing gate (standing, non-negotiable)

The agent NEVER publishes, posts, comments, or DMs on any platform autonomously. It drafts, stages, adapts, proposes timing windows, and monitors. Mitchell presses send.

**Why:** authenticity is the entire brand on every one of these 9 surfaces; one detected-automated post on HN or Reddit does permanent damage. Same principle as career-ops' "never submit an application without review."

Exception: GitHub pushes to content-companion repos are allowed (code artifacts, not social speech), on branches, with Mitchell merging.
